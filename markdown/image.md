## Image

[pl](pl/image.md)

In Markdown (MD), inserting an image is very simple — you use syntax similar to a link:

```
![alt text](path_or_URL)
```

### Examples

**1. Image from the internet:**

```
![cat](https://example.com/cat.jpg)
```

**2. Image from a local folder:**

```
![logo](./images/logo.png)
```

**3. Image with a title (tooltip):**

```
![dog](https://example.com/dog.jpg "This is a dog")
```

---

### How it works

* `!` indicates that it’s an image
* `[alt text]` – description (important for SEO and accessibility)
* `(path)` – link to the file

---

### Tips

* Relative paths (`./`, `../`) work, for example, on GitHub
* Make sure the file exists and the path is correct

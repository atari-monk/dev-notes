## Image

[en](../image.md)

W Markdown (MD) wstawienie obrazka jest bardzo proste — używasz składni podobnej do linku:

```
![tekst alternatywny](ścieżka_lub_URL)
```

### Przykłady

**1. Obrazek z internetu:**

```
![kot](https://example.com/kot.jpg)
```

**2. Obrazek z lokalnego folderu:**

```
![logo](./images/logo.png)
```

**3. Obrazek z tytułem (tooltip):**

```
![pies](https://example.com/pies.jpg "To jest pies")
```

---

### Jak to działa

* `!` mówi, że to obrazek
* `[tekst alternatywny]` – opis (ważny np. dla SEO i dostępności)
* `(ścieżka)` – link do pliku

---

### Tipy

* Ścieżki względne (`./`, `../`) działają np. w GitHubie
* Upewnij się, że plik istnieje i ścieżka jest poprawna
